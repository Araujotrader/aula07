html: Define o início e o fim de um documento HTML.

head: Contém informações do cabeçalho do documento, como título e links para arquivos CSS ou JavaScript.

body: Contém o conteúdo principal do documento.

h1, h2, h3, h4, h5, h6: São utilizadas para criar títulos e subtítulos.

p: Define um parágrafo de texto.

a: Cria um link para outra página ou recurso.

img: Insere uma imagem no documento.

ul, ol, li: São utilizadas para criar listas não ordenadas (com marcadores) e listas ordenadas (com números ou letras).

div: Cria um bloco de conteúdo e é frequentemente usado para agrupar elementos e aplicar estilos CSS.

span: Define uma pequena seção de texto dentro de um elemento.

form, input, button: São usadas para criar formulários interativos.

header, nav, footer, section, article: São exemplos de tags

font-family: Define a família de fontes a ser usada.

font-size: Define o tamanho da fonte.

background-color: Define a cor de fundo de um elemento.

margin, padding: Controlam o espaçamento ao redor do elemento.

border: Define a borda de um elemento.

width, height: Controlam as dimensões do elemento.

display: Define como o elemento deve ser exibido.

position: Controla o posicionamento do elemento na página.

float: Controla o posicionamento lateral do elemento em relação a outros elementos.

text-align: Define o alinhamento do texto.

list-style-type: Define o estilo de marcadores em listas.

transition: Cria efeitos de transição suaves para propriedades CSS.

box-shadow: Cria uma sombra ao redor de um elemento.
